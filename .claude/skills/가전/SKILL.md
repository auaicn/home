# 혼수 가전 견적 관리 ⚡

신혼부부를 위한 대형 가전제품 및 소형 가전의 견적을 관리하고 비교합니다.

## 주요 명령어

### `/가전 list`
모든 가전 제품 목록 표시
```
🏠 혼수/가전 (대형 가전)
현재 등록된 제품:
- 냉장고
- 세탁기
- 건조기
- 에어컨

🔌 혼수/소형가전 (소형 가전)
- 드라이기
- 청소기
- 공기청정기
```

### `/가전 search [키워드]`
키워드로 견적 검색
```
/가전 search LG 냉장고
→ LG 냉장고 관련 견적 3개 검색됨

/가전 search 세탁기
→ 세탁기 관련 견적 5개 검색됨 (가격: 1,200,000 ~ 4,500,000 KRW)
```

### `/가전 compare [제품명]`
같은 제품의 여러 업체 견적을 가격 및 스펙으로 비교
```
/가전 compare 냉장고

┌─────────────┬──────────┬────────┬─────────────┐
│ 업체        │ 모델명   │ 가격   │ 용량        │
├─────────────┼──────────┼────────┼─────────────┤
│ 삼성전자    │ 비스포크 │2,500K │ 870L        │
│ LG 전자     │ 디오스   │2,800K │ 900L        │
│ 대우전자    │ 프리스타 │1,800K │ 800L        │
└─────────────┴──────────┴────────┴─────────────┘

💡 가격순 정렬: 대우 < 삼성 < LG
🎯 성능순 정렬: LG (900L) > 삼성 (870L) > 대우 (800L)
```

### `/가전 add [제품명]`
새로운 견적 추가 (대화형)
```
/가전 add 냉장고

스텝 1: 업체 정보
- 업체명: 삼성전자
- 웹사이트: https://www.samsung.com/sec/

스텝 2: 제품 정보
- 모델명: 비스포크 4도어 냉장고 RF85C901301
- 가격: 2500000
- 통화: KRW
- 제품 URL: https://www.samsung.com/sec/refrigerators/...

스텝 3: 상세 스펙
- 용량: 870L
- 색상: 글램 네이비
- 에너지등급: 1등급
- 특징: 독립식 김장실

✅ 견적이 추가되었습니다! (ID: quote-xxx)
```

## 가전 카테고리별 가이드

### 🌡️ 대형 가전 (혼수/가전)

#### 냉장고
- **용량**: 800~900L (신혼가구 기준)
- **타입**: 4도어, 프렌치도어, 양문형
- **주요 업체**: 삼성, LG, 대우, 위니아
- **가격대**: 1.5M ~ 3M KRW
- **선택 팁**: 장기 A/S, 에너지 효율 확인

#### 세탁기
- **용량**: 8~10kg (2인 기준)
- **타입**: 드럼식 (추천), 통식
- **주요 기능**: AI 세탁, 에코 모드
- **가격대**: 1.2M ~ 2.5M KRW
- **선택 팁**: 건조 기능 함께 고려

#### 건조기
- **타입**: 열펌프식 (효율 좋음), 응축식
- **용량**: 8~9kg
- **가격대**: 1.5M ~ 2.5M KRW
- **선택 팁**: 세탁기와 같은 브랜드 추천

#### 에어컨
- **냉방능력**: 18,000BTU ~ 25,000BTU
- **타입**: 벽걸이, 스탠드형
- **가격대**: 0.8M ~ 1.5M KRW (1대)
- **선택 팁**: 에너지 효율, A/S 기간 확인

### 🔌 소형 가전 (혼수/소형가전)

#### 드라이기
- **와트**: 1400~2000W
- **주요 기능**: 음이온, 냉풍, 지표면 온도 제어
- **가격대**: 50K ~ 300K KRW
- **추천 제품**: 다이슨(프리미엄), 다인(가성비)

#### 청소기
- **타입**: 무선(추천), 유선
- **흡입력**: 200~350AW
- **가격대**: 300K ~ 1.5M KRW
- **선택 팁**: 필터 세척 용이성, 배터리 지속력

#### 공기청정기
- **정화능력**: 25~60평형
- **필터**: HEPA 필터, 활성탄 필터
- **가격대**: 200K ~ 1M KRW
- **선택 팁**: 초미세먼지(PM2.5) 제거율 확인

## 견적 추가 시 필수 정보

### ✅ 필수 필드:
- **업체명**: 정확한 한글명 (예: 삼성전자, LG 전자)
- **모델명**: 완전한 모델명
- **가격**: 원화 (KRW) 기본, 할인가 있으면 원래 가격도 기록
- **제품 URL**: 해당 제품의 정확한 링크
- **상세 스펙**: 용량, 색상, 에너지등급 등

### 📝 추가 정보:
- **할인 정보**: 혼수패키지, 계절 할인 등
- **A/S 정보**: 무상 A/S 기간
- **배송 정보**: 배송비, 설치비
- **특별 사항**: 여유금, 증정품 등

예시:
```json
{
  "vendor": "삼성전자",
  "productModel": "비스포크 4도어 냉장고 RF85C901301",
  "price": 2500000,
  "currency": "KRW",
  "specifications": {
    "용량": "870L",
    "색상": "글램 네이비",
    "에너지등급": "1등급"
  },
  "notes": "혼수패키제 10% 할인, 무상 A/S 5년"
}
```

## 혼수 가전 쇼핑 팁 💡

1. **혼수패키지 활용**
   - 대부분의 가전 업체는 신혼부부 혼수패키지 제공
   - 보통 10~20% 할인
   - 여러 제품 동시 구매 시 추가 할인 가능

2. **A/S 기간 확인**
   - 대형 가전은 5년 이상 A/S 보장 제품 추천
   - 부품 가용성이 오래 유지되는 대형 업체 선택

3. **에너지 효율 등급**
   - 1등급이 가장 좋음 (전기세 절감)
   - 20년 이상 사용하는 제품이므로 중요

4. **배송 및 설치**
   - 대형 가전은 배송비 + 설치비 함께 확인
   - 기존 제품 처리 비용 확인

5. **시즌별 가격 변동**
   - 여름: 에어컨, 냉장고 가격 상승
   - 겨울: 건조기 가격 상승
   - 명절: 혼수 수요로 가격 상승

6. **구매처 비교**
   - 대형마트: 여유금 + 할인
   - 공식 온라인: 최신 모델
   - 중소 판매점: 추가 할인 가능
